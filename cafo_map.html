Presence of CAFOs in Socially Vulnerable Areas of North Carolina
UNC-Chapel Hill 2023

Created as part of a geospatial analysis course at UNC-Chapel Hill, these maps and graphs come from a project analyzing whether feedlots (CAFOs) in North Carolina are more concentrated in socially vulnerable areas. The CAFO data from the NC DEQ while the social vulnerability data came from the CDC's social vulnerability index. After finding the datasets, I prepared each dataset for analysis using Microsoft Excel and R by removing superfluous data, adding additional information, and ensuring that each dataset was structured the same to join them. The maps were then completed with analysis in ArcGIS Pro.
