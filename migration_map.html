Mapping European Public Opinion of Migration
UNC-Chapel Hill 2024

This map was created as part of a project for a web mapping class. This Leaflet-based map shows the relationship between Europeans' public opinion of migration to the EU and the actual number of asylum seekers for the years 2014-2023.
